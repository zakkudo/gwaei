Description

Used for storing and loading parsed dictionary information from cachefiles. This is done by serializing/deserializing each of the internal LwParsed and LwIndexed objects. Usually a different LwDictionaryCache will be created for each normalized dictionary form used, and are indexed into a tree indexed by its LwUtf8Flags.
