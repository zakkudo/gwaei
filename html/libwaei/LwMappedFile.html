Description

This is a wrapper around GMappedFile with extra convenience methods and it also stores the path of the mapped file.

lw_mappedfile_set_writable ()

void
lw_mappedfile_set_writable (LwMappedFile *self,
                            gboolean writable);

Changes the writable flag on the underlying GMappedFile by unreferencing it and replacing it with a new GMappedFile. This any buffer that was loaded with lw_mappedfile_get_contents() is no longer valid and will have to be refetched.
