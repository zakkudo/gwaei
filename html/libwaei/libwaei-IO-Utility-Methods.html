lw_io_fwrite ()

gsize
lw_io_fwrite (FILE *stream,
              const gchar *TEXT,
              gint length,
              LwProgress *progress);

A convenience method to write a string to a file while reporting progress. The chunk size defaults to the page file size of your system, which can be overridden by setting prefered chunk size on the LwProgress object.

Parameters

stream

FILE to write to

  

TEXT

Text to write to the stream

 

length

The length of TEXT in bytes or 0 to have it calculated

  

progress

A LwProgress to track progress or NULL to ignore it

  

Returns

The number of bytes written
